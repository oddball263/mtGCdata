====================================================================
Johns Hopkins - Getting and Cleaning Data, Course Project

====================================================================

Tidy dataset created from Human Activity Recognition 
Using Smartphones Dataset
Version 1.0.

Data collected with accelerometers from Samsung Galaxy S smartphone

M. Tato, 2016_04_03
====================================================================


Provided for each record:
=========================================
The average of each variable measurements on the Mean and Standard Deviation:
 - For each Activity and each Subject.


The dataset includes the following files:
=========================================

- 'README.txt'

- 'CodeBook.RMD': Details on dataset and cleaning outline.
                  Also in .MD and .html

- 'run_analysis.R': R code for transforming raw data into tidy dataset.

- 'tidy_groupedData.txt': Tidy dataset.
